My Skills

In Web Design

HTML

HTML (HyperText Markup Language) is the standard language used to create the structure and content of web pages. It defines elements like headings, paragraphs, images, and links, serving as the foundation for all websites.

CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls the appearance of HTML elements—such as colors, fonts, spacing, and positioning—allowing developers to create visually appealing and responsive designs.

JAVASCRIPT

JavaScript is a scripting language used to add interactivity to web pages. It enables features like sliders, form validation, animations, and dynamic content updates without reloading the page.

Java is a versatile, object-oriented programming language used for building applications across platforms. It’s commonly used in enterprise software, Android apps, and large-scale systems.

PYTHON

Python is a high-level programming language known for its simplicity and readability. It is widely used in web development, automation, data analysis, artificial intelligence, and more.

C

C is a powerful low-level programming language often used in system software, embedded systems, and performance-critical applications. It provides fine control over hardware and memory.
